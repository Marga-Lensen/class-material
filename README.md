## Overview

In this repository you will find:
- slides and notes from class
- setup instructions
- live-coding examples
- small React demo projects

The material is organized by milestone and will grow over time.

## Repository Structure

### `00-JS-recap_milestone0`
JavaScript recap material used before starting React.

### `01-React_example-projects`
Small standalone React example projects.

### `01-React_milestone1`
Slides and material for the first React milestone.

### `01-React_milestone2`
Material for props, state, `useEffect`, and API fetching.

## How to Use This Repo

Use this repository to:
- review what we covered in class
- revisit live-coding examples
- run the demo projects locally
- continue practicing between classes

## Running the React Projects

Each React project has its own `package.json`.

Inside a project folder, run:

```bash
npm install
npm run dev
```
