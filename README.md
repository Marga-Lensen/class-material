## How to Use This Repo

Use this repository to:
- review what we covered in class
- revisit live-coding examples
- run the demo projects locally
- continue practicing between classes

## Running the React Projects

Each React project has its own `package.json`.

Inside a project folder, run:

```bash
npm install
npm run dev
```
